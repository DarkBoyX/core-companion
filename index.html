Standalone Integration Interface

Connecting Standalone to Roblox Studio
  1. Install the Core AI Studio plugin (see roblox_plugin/CoreAI_Plugin.server.lua).
  2. Run the bridge server (server/server.js) and point both the plugin and this page at it.
  3. Open the plugin's panel in Studio — it shows a pairing code like 7K4P-XQ2M.
  4. Paste that code into the Roblox Studio Bridge box in the sidebar and click Connect.
  5. Ask Core to make a change ("create a red part") — it'll show up in your Studio workspace live.
Plugin Source
The full plugin lives in roblox_plugin/CoreAI_Plugin.server.lua in the project bundle — it creates the pairing session, shows the dock widget UI, polls the bridge server, and runs a whitelisted set of safe actions (create_part, set_color, set_size, set_position, delete_part, delete_all). It never executes arbitrary code sent from the web.
