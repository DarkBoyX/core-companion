How it works
  1. Install the Core AI Studio plugin (see roblox_plugin/CoreAI_Plugin.server.lua).
  2. Run the bridge server (server/server.js) and point both the plugin and this page at it.
  3. Open the plugin's panel in Studio — it shows a pairing code like 7K4P-XQ2M.
  4. Paste that code above and click Connect.
  5. Ask Core to build something ("make a settings menu UI", "add a leaderboard module script") — it'll show up live in Studio.
What Core Can Build
✓ create_folder — organize ReplicatedStorage / ServerScriptService / StarterGui / StarterPlayerScripts
✓ create_script — Script, LocalScript, or ModuleScript, with generated source
✓ create_ui — ScreenGui, Frame, TextLabel, TextButton, and other whitelisted GUI classes
✓ delete_instance — remove something Core (or you) created earlier
✗ no parts, meshes, unions, or any 3D geometry — that whitelist is intentionally absent from the plugin.
